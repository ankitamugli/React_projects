## 🚀 Getting Started

### Prerequisites

* Node.js
* npm

### Installation

1. Clone the repository

```bash
git clone https://github.com/ankitamugli/React_projects.git
```

2. Navigate to the project folder

```bash
cd React_projects
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

or

```bash
npm start
```
